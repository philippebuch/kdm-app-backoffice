Howto

Presentation

FrontOffice uikit is a tool for web developers. It provides documentation for the following front-end topics:

  • Masterpage (site structure, header, footer, menus).
  • Layout (page structure).
  • Components, with code samples and usage description.
  • Pages, fully working as they should appear on production.

The package files

  • src/
    • fonts/
    • icons/ fonticons src, icomoon set icons in selection.json
    • js/un-concatenated vendor js files
    • img/
    • less/ Less (Leaner Style Sheets) CSS prepocessor files
    • views/
  • build/
    • css/app styles, compressed and uncrompressed
    • fonts/
    • icons/ builded fonticons
    • img/
    • js/app scripts, compressed and uncrompressed
    • uikit/
      • css
      • js
    • HTML pages on root.

Usage

Navigation

  • In every uikit pages, you will find a table of content menu in the left side panel.
  • Components are displayed within tabs panels, one for preview, one for code snippet.

Getting HTML code

It is recommended to get snippet HTML code by clicking the button « Copy code to clipboard ».
Never copy the code from the browser page source code, as it might contain generated code.
